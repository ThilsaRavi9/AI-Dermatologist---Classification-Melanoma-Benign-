About the Project

Abstract

This project explores the integration of computer vision techniques in healthcare using deep convolutional neural networks to differentiate malignant melanoma from benign melanoma. By utilizing a dataset of dermoscopic images from the ISIC Archive, we build a convolutional neural network using TensorFlow (Keras) to provide rapid and cost-effective diagnosis for melanoma. The model's performance is evaluated based on accuracy, F1 score, and other metrics. The study also discusses the scalability of the model to other skin problems and presents a web app interface for accessible use.

Key Features

  • Utilizes deep learning for melanoma classification
  • Trained on a dataset of 3,297 dermoscopic images
  • Provides rapid and cost-effective initial diagnosis
  • Web-based interface for easy access and use

The AI Dermatologist project aims to revolutionize melanoma diagnosis through the application of deep learning techniques. By leveraging convolutional neural networks and a comprehensive dataset of dermoscopic images, we've developed a system capable of differentiating between benign and malignant melanomas with high accuracy.

Our model, built using TensorFlow and Keras, has been trained on 1,800 images of benign moles and 1,497 images of malignant moles from the ISIC Archive. This extensive dataset allows our AI to recognize subtle patterns and features that might be overlooked in traditional visual inspections.
